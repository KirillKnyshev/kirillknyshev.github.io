Cyclistic Case Study

Completed project by Kirill Knyshev

Summary

You examined Cyclistic bike share data to show how casual riders and members behave. Your goal was to support a plan to grow annual memberships. You used spreadsheets, SQL and Tableau. You focused on patterns in ride length, bike types, stations and ride timing.

Scenario

Cyclistic serves riders in Chicago. Casual riders buy single rides or day passes. Members pay for a yearly plan. Members bring in more revenue, so leadership wants more riders to switch to yearly plans. Your task was to review past rider behavior and support the marketing plan.

Business task

Find clear patterns in how members and casual riders use Cyclistic bikes. Show the differences with data. Present insights that help the team target casual riders with a strong message.

Data sources

You used twelve CSV files with one year of ride data. Each record included ride time, bike type, start station, end station and rider type. You also used public bike station data from the City of Chicago. All data was structured and ready for analysis after cleaning.

Data cleaning steps in Excel

  • Loaded all twelve monthly CSV files.
  • Created started_at_full and ended_at_full as datetime fields.
  • Created a ride_length field by subtracting end from start.
  • Formatted ride_length as time (HH:MM:SS).
  • Saved each cleaned file as a new CSV.

Data cleaning steps in SQL

  • Uploaded all cleaned files to BigQuery.
  • Built one combined table using UNION ALL.
  • Removed errors and blank fields.
  • Created fields for day of week and ride length in seconds.

Analysis results

Ride length

  • Casual riders ride longer on average.
  • Members ride shorter but more frequent trips.
  • Max ride times are similar for both groups.

Day of week behavior

  • Members peak on Tuesday and drop on Sunday.
  • Casual riders peak on Saturday.
  • Both groups have longer rides on weekends.

Bike type

  • Both groups use electric bikes more often.
  • Classic bikes have longer ride times for both groups.

Stations

You identified start stations with high use for each group. These locations support targeted marketing.

Insights

  • Casual riders ride longer and ride more on weekends.
  • Members use bikes more often on weekdays.
  • Both groups choose electric bikes often.
  • Station patterns show strong commuter behavior for members.

Conclusion

The data shows clear differences between casual riders and members. These patterns help shape ads and messages for casual riders. The marketing team can build a focused strategy with these insights.
